About ESOBuild.com

ESOBuild.com is an up-to-date report of "meta" Elder Scrolls Online trial builds, automatically curated from the top 10 ranked ESO Logs reports for each trial. It lets you explore builds used by top-performing players for each trial and boss encounter. In short, it answers the question: "what is meta right now?"

How It Works

To this tool, a "build" is a unique combination of subclasses (skill lines), and two "5-piece" gear sets with 4 or more pieces of each equipped. Yes, this is a simplifying assumption. Other heuristics may be added later other (hello 5/2/2/2/1 builds?) build patterns.

ESO Build scans the top-ranked trial reports that have logs using the ESO Logs API. Player gear and subclass information is extracted from each boss fight to identify the most frequently-seen combinations: the meta. This approach might not end up selecting the build of the highest DPS if they have a unique build. It will find common builds. Also note that ESOBuild.com is way better at finding DPS builds than healers and tanks. There are lower numbers and of healers and tanks, and a proliferation of subclass options makes it hard to select common builds from a small sample. However, esologs.com links are provided so you can explore group composition at esologs.com and dig out more unique builds.

Build Requirements

A build is considered "common" and included in our report if it appears:

  • 5 or more times for DPS builds
  • 3 or more times for healer and tank builds

This ensures we only show builds that have been seen used by multiple players for that fight across the 10 top-ranked reports.

Data Source

All build data comes from ESO Logs, the premier log analysis site for Elder Scrolls Online. If you are uploading encounter logs to esologs.com, I recommend using ESO Log Tool by sheumais for better upload speed than the default client, and helpful features like script enumeration for scribed abilities.

Updates

Trial builds are updated from esologs.com once every 14 hours, with a different trial processed each hour. Trials are processed in order from oldest: AA, to newest: OC. The site is hosted in GitHub Pages and proxied by Cloudflare, so content is agressively cached. The footer in the home page shows the last updated time in your timezone.
